# IT-Expenditure-Analysis
# Overview
This project analyzes IT expenditure performance by comparing Actual, Forecast, and Planned spending using Power BI.
The goal is to identify budget variances, understand cost drivers, and support better IT financial planning and control.

# Business Objective
-Monitor IT spending across time, regions, and IT areas
-Identify overspending and underspending against plan
-Improve budgeting accuracy and financial visibility

# Dataset
Monthly IT expenditure data including:
-Date (month-level)
-Region, Business Area, IT Area
-Cost Element Groups
-Actual, Forecast, and Planned spend
-Future periods contain forecast and plan values, while actuals are available only for completed months.

# Key Features
-KPI cards for Actual, Forecast, Plan, and Variance %
-Trend analysis (Actual vs Forecast vs Plan)
-Variance analysis by IT Area
-Cost distribution by cost element
-Interactive slicers for dynamic filtering

# Tools Used
-Power BI Desktop
-Power Query
-DAX
-Excel

# Outcome
The dashboard provides clear visibility into IT spending patterns and variance drivers, helping stakeholders make informed budgeting and cost optimization decisions.
